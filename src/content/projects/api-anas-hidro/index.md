API ANAS Hidro e uma API em TypeScript com Node.js, Docker, MongoDB e autenticacao por token para inserir, atualizar, remover e consultar dados complexos de medicao.

Ela conversa diretamente com a parte do meu trabalho ligada a telemetria, dados de campo, validacao, persistencia e integracoes recorrentes.

## O que este projeto demonstra

- API REST em TypeScript.
- Persistencia em MongoDB.
- Autenticacao baseada em token.
- Organizacao de dados de medicao e hidrologia.
- Base para pipelines de leitura, transformacao e envio.
