ClimaTempo e um projeto full stack com frontend em React e backend em Python/FastAPI para consulta de previsao do tempo com uma interface simples e moderna.

Mesmo sendo um projeto menor, ele demonstra um ciclo completo: consumo de API externa, backend proprio, frontend separado, tratamento de dados e apresentacao para usuario final.

## O que este projeto demonstra

- Backend com FastAPI.
- Frontend em React e TypeScript.
- Integracao com dados externos.
- Separacao entre API e interface.
