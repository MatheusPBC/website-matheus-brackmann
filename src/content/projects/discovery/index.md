Discovery e um MVP de descoberta arquitetural para apoiar iniciativas de modernizacao de sistemas.

O projeto analisa repositorios locais a partir de codigo, infraestrutura como codigo e CI para gerar um artefato inicial com stacks detectadas, entrypoints relevantes, pontos de integracao, riscos visiveis e evidencias rastreaveis.

## O que este projeto demonstra

- API FastAPI com endpoint de descoberta.
- Arquitetura modular com camadas de aplicacao, dominio e infraestrutura.
- Deteccao inicial para Python, Node, Go, Terraform e GitHub Actions.
- Exportacao opcional de artefatos estruturados em JSON.
- Testes de unidade e integracao para validar o fluxo principal.
